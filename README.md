# 🌌 StorySpot

> A quiet place on the internet where strangers feel understood.

StorySpot is an anonymous storytelling platform where users share personal experiences on a public wall and are instantly matched with someone who has felt something similar. Instead of likes, followers, or noise — it focuses on **real human connection through shared emotion**.

---

## ✨ Core Idea

Most platforms reward attention.  
StorySpot rewards **honesty**.

- You write a story  
- The system finds someone who *feels the same*  
- You connect privately — without ever knowing who they are  

No profiles. No followers. No pressure.

---

## 🧱 Features

### 🌍 Public Story Wall
- Post short anonymous stories
- Explore a global feed of raw, unfiltered thoughts
- No likes, no comments — just stories

---

### 🤝 Smart Matching
- Instantly matched with one person who shared a similar story
- Lightweight semantic/keyword-based similarity
- Each match creates a **private 1:1 connection**

---

### 🏠 Private Home
- Your personal, private space
- Contains multiple “walls” — one per connection
- Only you can access it

---

### 📝 Connection Walls
- Anonymous note exchange between two users
- Simple, chronological messages
- No typing indicators, no read receipts

---

### 💭 Emotional Experience
- Reinforcing messages like:
  > “You’re not alone in this.”
- Designed to feel calm, safe, and human

---

## 📈 Growth Features

- 📤 Shareable story cards (for social media)
- 🧠 Daily writing prompts
- ❤️ “Someone feels this too” feedback loop
- 🔔 Subtle new message indicators

---

## 💰 Monetization (Optional Layer)

- Freemium match limits
- Unlock old connections
- Priority matching
- Deep conversation mode
- Anonymous appreciation tokens

---

## 🎯 Design Philosophy

StorySpot is built on a few strong principles:

- **Anonymity over identity**
- **Depth over engagement metrics**
- **Calm over stimulation**
- **Connection over consumption**

---

## 🧠 How It Works (Flow)

User enters app
   ↓
Writes a story
   ↓
System finds similar story
   ↓
Private connection created
   ↓
Users exchange notes anonymously

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
